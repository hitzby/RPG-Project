# ⚔️ Aincrad — 塔层爬行 RPG

> "Link Start！欢迎来到 Aincrad！"

一款基于 C++17 的命令行 Roguelike RPG。选择职业，爬上第 100 层，挑战 Boss，在随机词缀、掉落装备和技能组合中找到属于你的通关路线。

## 游戏特性

- **100 层塔爬行**——每 10 层遭遇强化 Boss，通关第 100 层获胜
- **3 职业 × 5 技能**——战士、法师、刺客各有独特技能树和成长路线
- **7 种怪物词缀**——强力、坚韧、迅捷、剧毒、治疗、爆炸、护盾，随楼层叠加
- **职业专属装备**——武器/防具按职业分类，刺客无法装备法杖
- **四档品质掉落**——普通、精良、稀有、史诗，高层品质下限自动提升
- **随机事件**——宝箱、陷阱、游商、温泉、神社，每层都有意外惊喜
- **存档/读档**——随时保存，下次继续

## 快速开始

```
=== Link Start! 欢迎来到 Aincrad! ===
请输入你的名字：Kirito
选择职业：[1] 战士  [2] 法师  [3] 刺客
```

进入每层后可选择：

| 选项 | 说明 |
|------|------|
| `[1]` 战斗 | 触发随机事件后与本层怪物战斗 |
| `[2]` 背包 | 查看、使用、装备或丢弃物品 |
| `[3]` 装备 | 查看当前装备/卸下装备 |
| `[4]` 退出 | 退出前可选择保存 |
| `[5]` 存档 | 随时保存当前进度 |
| `[6]` 查看状态 | 查看角色详细属性 |

战斗胜利后可选择继续刷怪、前往下一层、打开商店或升级技能。

## 技能系统

每个职业拥有 5 个技能，随等级逐步解锁。每回合回复 **8 MP**，合理安排技能使用是通关关键。

### 战士技能

| # | 名称 | 解锁 | CD | MP | 说明 |
|---|------|------|----|----|------|
| 1 | **猛击** | Lv.1 | 2回合 | 6 | 稳定单体伤害，主力输出 |
| 2 | **战吼斩** | Lv.8 | 3回合 | 14 | 伤害 + 回复 15% 最大HP |
| 3 | **盾反** | Lv.18 | 4回合 | 18 | 削减敌方攻击力 35%（对强力词缀效果显著） |
| 4 | **血怒** | Lv.28 | 4回合 | 22 | HP>60% 时×1.0；HP 30~60% 时×1.8；HP<30% 时×3.0 |
| 5 | **无双斩** | Lv.40 | 6回合 | 35 | 终极大招，极高伤害 |

### 法师技能

| # | 名称 | 解锁 | CD | MP | 说明 |
|---|------|------|----|----|------|
| 1 | **火球术** | Lv.1 | 2回合 | 10 | 稳定火焰单体 |
| 2 | **冰霜新星** | Lv.8 | 3回合 | 18 | 伤害 + 冻结削减敌方攻击 30%（强力词缀加倍） |
| 3 | **毒云术** | Lv.18 | 3回合 | 0 | 特殊：**不消耗 MP，反而回复 30 MP**，同时造成毒素伤害（无视护盾） |
| 4 | **奥术爆发** | Lv.28 | 5回合 | 当前MP的50% | 消耗当前 MP 一半，伤害 = 消耗量 × 3（MP 越满越强） |
| 5 | **流星陨落** | Lv.40 | 6回合 | 50 | 极高伤害，敌方 HP < 30% 时追加 30% 当前HP补刀 |

### 刺客技能

| # | 名称 | 解锁 | CD | MP | 说明 |
|---|------|------|----|----|------|
| 1 | **穿刺** | Lv.1 | 2回合 | 8 | 低CD快速刺击，主力输出 |
| 2 | **影袭** | Lv.8 | 3回合 | 16 | **必定暴击 ×2.5**，护盾效果减半 |
| 3 | **连环刺** | Lv.18 | 4回合 | 24 | **随机连击 2~4 次**，首击受护盾，后续穿透 |
| 4 | **吸血刃** | Lv.28 | 4回合 | 28 | 伤害 + 吸收造成伤害的 40%~60% 为 HP（随技能等级提升） |
| 5 | **瞬杀** | Lv.40 | 6回合 | 40 | 无视护盾，造成敌方**当前 HP 的 50%**（Boss 状态减半为 25%） |

> 技能可在战后菜单 `[8] 技能` 中花费金币升级，每级基础伤害 +15，MP 消耗 +3。

## 装备系统

### 品质分级

| 品质 | 概率（低层） | 颜色感 | 特点 |
|------|------------|--------|------|
| 普通 | 50% | 白 | 基础属性 |
| 精良 | 30% | 蓝 | 属性提升约 1.5× |
| 稀有 | 15% | 紫 | 属性提升约 2.5×，耐久更高 |
| 史诗 | 5% | 橙 | 顶级属性，每 20 层品质下限 +1 |

### 职业限制

所有武器和大部分防具都有职业限制，尝试装备其他职业的装备会直接拒绝。少量**通用防具**（护符/腰带）任何职业均可穿戴。

### 丢弃与变现

背包中无法使用的装备可以选择**丢弃**，获得售价 1/3 的金币作为补偿。在宝箱事件拾取时也可直接丢弃。

## 怪物与词缀

### 成长曲线

| 楼层 | 怪物 HP（约） | 怪物攻击（约） |
|------|-------------|--------------|
| 1 | 43 | 6 |
| 10 | 198 | 26 |
| 20 | 355（Boss×3） | 46 |
| 50 | 650 | 79 |
| 100 | 1410（Boss×3）| 154 |

> 每 10 层出现 Boss：血量×3，攻击×2，经验×4，金币×5。

### 词缀系统

词缀随楼层增加触发率（1层约40%，100层约70%），高层怪可叠加多个词缀。

| 词缀 | 概率 | 效果 |
|------|------|------|
| **强力** | 25% | 攻击力 + 等级×3 |
| **坚韧** | 20% | HP + 等级×25 |
| **迅捷** | 15% | 闪避率提升（上限45%） |
| **剧毒** | 13% | 每回合对玩家造成毒素伤害 |
| **治疗** | 12% | 每回合自我回血 |
| **爆炸** | 8% | 死亡时对玩家造成爆炸伤害 |
| **护盾** | 7% | 每次受击吸收固定伤害 |

## 随机事件

每层战斗前有概率（30%~80%）触发随机事件：

| 事件 | 效果 |
|------|------|
| 🎁 **宝箱** | 随机金币 + 随机物品，可选择入包/装备/丢弃 |
| 🪤 **陷阱** | 受到随机伤害（约当前 HP 的 8~15%） |
| 🧙 **游商** | 随机5件商品，可购买、刷新（50金）|
| 🌸 **温泉** | 回复大量 HP 和 MP |
| ⛩️ **神社** | 随机结果：获得大量经验和金币 / 损失少量 HP 和 MP / 无事发生 |

## 存档系统

- 游戏在退出前、战后菜单均可手动存档
- 存档记录楼层、角色属性、背包和已装备物品
- 旧版存档（无职业限制字段）读取时自动兼容，装备视为"通用"

## 项目结构

```
.
├── main.cpp          # 主循环、物品生成、战斗逻辑、随机事件
├── character.h/cpp   # 基类 Character：name/hp/attack/mp
├── enemy.h/cpp       # 敌人类：词缀系统、掉落奖励
├── player.h/cpp      # 玩家类：背包/装备/技能管理
├── jobs.h/cpp        # 职业派生：Warrior / Mage / Assassin
├── skills.h/cpp      # 技能系统：15个技能的实现
├── item.h/cpp        # 物品类：品质/职业限制/耐久
├── save.h/cpp        # 存档系统：序列化/反序列化
└── CMakeLists.txt    # 构建配置
```

### 类继承关系

```
Character
├── enemy
└── player
    ├── Warrior
    ├── Mage
    └── Assassin
```

## 编译说明

**要求**：CMake 3.10+，支持 C++17 的编译器（GCC / Clang / MSVC）

```bash
# 创建构建目录
mkdir build && cd build

# 配置
cmake ..

# 编译
cmake --build .

# 运行
./RPG_Game
```

> **注意**：如果遇到链接错误 `undefined reference to SaveSystem::...`，请确认 `CMakeLists.txt` 中 `add_executable` 包含了 `save.cpp` 的实际路径。
